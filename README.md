# Hedge Fund Pod Dashboard

This project is a high-fidelity institutional dashboard designed to visualize the outputs of the **Hedge Fund Pod** multi-agent simulation.

It maps exactly how five institutional frameworks (Quant, Pod, Macro, Fundamental, Activist) handle the "Bond Vigilante" market shock using structural constraints like the Deafness Protocol, explicit Cost of Carry parameters, and competitive Crowding penalties.

### Deployment Note
This repository is packaged as a static front-end artifact completely devoid of local dependencies. It utilizes CDN-hosted Tailwind CSS and Chart.js, making it intentionally lightweight and immediately deployable.

It is structured and intended for direct deployment to **GitHub Pages**.
